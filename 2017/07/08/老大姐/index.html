老大姐

发表于 2017-07-08 | |

今天中午去食堂打饭的时候，我终于不得不承认，老大姐不会再在那里出现了，她真的已经退休了。

我不知道老大姐姓什名谁，我们之间唯一的关联是：中午我会去食堂吃饭，而她之前负责在食堂打米饭。现在细想起来，我们甚至连招呼都没有打过。但是她的热情，她的麻利，她的自来熟，都给我留下了很深的印象。

退休之前，老大姐每天都是中午去食堂吃饭必不可少的一道风景线。

每天到食堂之后，都能看见她在自己的岗位上忙碌着。不管在她前面排队的人是多不是少，她永远是一张笑脸在那里叫着，双手不停挥舞。

一般来说，她的整套流程是这样子的：她左手迅速在刷卡机上按下五毛，或者七毛，或者一块，然后顺势接过你的碗，右手的小铁铲已经铲起一块或者一块半米饭，倒入你的碗中——米饭已经被提前分割成齐整的小方块了。同时，她会根据当时那一板米饭的软硬、厚薄，再给你找补个三分之一块或者二分之一块。等你刷完卡，盛好米饭的碗已经又递回到你跟前，等着你伸手去接呢。

这时她也并未闲着，她那和善的双眼已经开始打量下一个排队的人，判断着TA要吃多少钱的米饭。等你接过碗，才一转身，甚至都还没迈步，老大姐已经又熟练地在刷卡机上按下了五毛或者七毛，或者一块了。那预示着下一个碗里将要打入的米饭的数量。

整个过程行云流水，没有一丝多余，也没有一点拖沓，一切都刚刚好。而老大姐在整个过程中所表现出来的那份热忱、专注和敬业，不输任何一个在手执画笔的艺术家。

工匠精神！每当这个时候，我的心里总会冒出这个词。

很少有人对她按下的数字表示异议。这主要归功于两个原因，一个是老大姐记性特别好。每天在这个食堂吃饭的至少有五六百人，但只要你去过四五次之后，她就能记住你要吃多少钱的米饭，八九不离十。二一个是职业素养。几十年如一日坚守岗位，让她练就了一双火眼金睛，扫上一眼胖瘦轻重，她就能判断出来，你吃多少钱的米饭刚刚好。

当然了，也有其他情况存在，我正好就属于这比较尴尬的其他情况。由于稍微有点儿胖，老大姐总是觉得我应该吃七毛钱的米饭才够，但事实我吃五毛钱的米饭足够了。每当这时我就在想，她这么多年练就的记性和职业素养，怎么在我身上就是不起作用呢？

其实刚开始的时候，我也是有过挣扎的。在她每次说完“七毛是吧？”之后，我都会很不好意思地来这么一句“五毛，谢谢！”，然后她先消掉刷卡机上面的七毛，改成五毛，顺势接过我手里的碗，右手麻溜地铲起一整块的米饭倒入碗中，然后再递还给我。等我刷完卡，她已经将这一系列动作完成了。

在我坚持了好多次都无果之后，我可耻地屈服了——毕竟后面还有那么多人在排队，每天在我身上都要耽搁时间，这样不太好。还有一个更为重要的原因是，我这一耽搁，她的动作总要停顿一下，就变得不再连贯了。比起多吃两毛钱的米饭，这事让我更不能忍。

所以，后来画风便再度回归和谐：

“七毛是吧？”

“嗯！”

“好的，下一个”

然后我端着碗去打菜，老大姐双手依然不停地挥舞着。同时，她嘴里也没闲着。

“你吃五毛够了吧？”

“你吃七毛怎么样？”

“这盘米饭太薄了，来我再给你添点儿。”

“今天这米饭有点软，多吃一点儿才经饿！”

她声音虽大，但一点儿也不让人觉得厌烦，反面会让你产生排队并没有那么让人厌烦的幻觉。只要有她在，只要她的声音在响，只要她的双手在动，你就能够感觉到，很快你手里在碗就得装上热腾腾的米饭。很可能下一个到你了。

眼前正在给人打米饭的这个小姑娘，没有了老大姐的吆喝，也没有老大姐麻利的动作，动作迟缓、拖沓，脸上一副你欠她两百万的表情，我突然发现，排队确实是一件很让人恼火的事情，特别是在肚子饿特别快的下雨天。

唉，美好的事情又少了一件。

但又有什么关系呢，事情总会变好的。
