数学空格

在 LaTeX 的数学公式代码中, 空白字符仅仅被当做 分隔符, 而不会当做空白字符来处理, 数学公式不同符号之间的空白不会受到输入的空白字符的影响. 如果希望在数学公式中添加额外的空格, 则需要使用特殊的命令. LaTeX 中支持多种不同宽度的空格, 它们分别是

效果 数学空格 宽度
$a \quad b$ \quad 空格宽度是当前字宽(18mu)
$a\qquad b$ \qquad 空格宽度是36mu
$a\ b$ 在\后面敲一个空格 空格宽度是标准空格键效果
$a,b$ , 空格宽度是3mu
$a:b$ : 空格宽度是4mu
$a;b$ ; 空格宽度是5mu
$a!b$ ! 空格宽度是-3mu(向左缩)

数学字体

LaTeX 使用特殊的数学字体来排版数学公式. 一般而言, 数学公式的字体和正文的字体会有所区别. 不过, 用户也可以像修改正文字体一样修改数学字体. LaTeX 提供了如下几个命令来设置数学字体:

代码 意义
maths code 默认字体
\mathnormal{maths code} 默认字体
\mathrm{maths code} 正体
\mathit{maths code} 斜体
\mathsf{maths code} 无袖衬字体
\mathbf{maths code} 粗体
\mathcal{maths code} 手写体

字体大小

在 正文排版 中, 可以使用 \large 之类的命令改变正文字体的大小. 对于数学公式, 也有对应的改变数学公式字体大小的命令.

数学字体大小命令 说明
\displaystyle 显示数学公式的字体大小
\textstyle 行内数学公式的字体大小
\scriptstyle 上下标的字体大小
\scriptscriptstyle 上下标的上下标字体大小
