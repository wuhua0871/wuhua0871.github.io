你那么迷信你居然不知道！

发表于 2017-04-20 | |

1.什么是迷信？

对于像我一样，在上世纪80年代出生，90年代成长起来的人来说，迷信 一定不会是一个陌生的词汇，用如雷贯耳来形容它，是一点儿也不过分的。

虽然以我理解，迷信 应该是一个中性词，但是在我有限的三十几年生命里，却从来没有见过它被中性地使用过。没有，一次都没有过。

在我从童年到青少年那段时光里，迷信这个词更多地是和另外一个词 封建 以固定搭配 封建迷信 的形式出现的。要知道，在那个时候，破除封建迷信可是举国上下的重要任务呢。

随着破除封建迷信的任务的完成，现在，这个词组出现的频率已经远远低于那些年了。

（对，我这里想要表达的意思就是，既然破除封建迷信的任务已经完成了，那么现在仍然存在的那些当年被强烈反对过现象当然就不再是封建迷信了，人家现在有更高大上的名字，叫做“国粹”好么！连过去所谓的玄学，现在都改名叫做“国学”了，难道你没看到么！）

在一个不存在的网站 维基百科 上面，它是这样解释迷信的：

迷信（英语：Superstition）是对某一些事物迷惘而不知其究竟，但又盲目地相信其说。Encarta百科全书上对“迷信”的定义是：非理性地相信某种行为或仪规具有神奇的效力。

“迷信”的本质是对事理的陈述不抱怀疑态度，就像科学家说地球不是宇宙的中心，若只信其陈述，而不加考证，这就是一种对科学权威的迷信。

破除迷信必须经过科学的考证步骤，大胆假设，小心求证。科学本身与迷信、信仰或宗教有着重要区别。科学在本质上要求怀疑主义，并在实践中公开倡导怀疑主义。

怀疑主义要求分析型思维以及批评型思维，要求怀疑一切，即不相信任何未经证实的事物。对已经证实的事物，也要怀疑其时空的局限性及观察的准确性。未经证实的理论称作假说或猜想，而假说或猜想完全有可能被证明是错误的。错误的假说或猜想必须被抛弃。

在我看来，这个的阐释无疑是准确的。

它抓住了迷信两个最根本的特点：盲目 和 不怀疑。

2. 我们都很迷信！

看来呀，迷信不单单只有封建迷信一种。

凡是具有以下两个特征的行为和想法，我们都可以将它归结为迷信。这两个特征分别是：

  1. 盲目相信，
  2. 不抱怀疑态度。

这样来看，我们都是迷信的人，而且相当迷信。你猜到我要说什么了吗？

我猜你已经想到了。是的，我想要说的正是这个：今天，我们都迷信一个叫做科学的东西！ 而且，我们对科学的迷信都是自发的，发自内心，甚至可以说是无以复加的。

但这不是我们的错！

毕竟，是科学–准确地说是现代科学—支撑起了我们生活的方方面面。科学发展带来的技术进步，构成了我们文明进步的基石。从原始文明，到农耕文明，游牧文明，到工业文明，再到现代的后工业时代文明，都是科学在背后推动的。

我们生活，下至的衣食住行，吃喝拉撒，上到航空航天，全球定位，没有什么和科学扯不上关系。

如果没有科学，今天我们都不可能有互联网，也不会有方便快捷的网络社交。这样的话，我也就更不可能在这里啰啰唆唆比划半天文字，把自己这点儿粗鄙的认知记录下来，供大家消遣和品鉴，贻笑大方了。

我们都这么迷信的科学。但是，你想过没有：科学真的有那么“科学”吗？

3. 科学真的“科学”吗？

科学真的有那么“科学”吗？ 或者，我们换个问法：

科学真的是无懈可击的吗？
科学等同于真理吗？

都不用过多考虑，根据我们的经验–或者说我们中学时学过的马克思唯物主义–，凡是这样说的很绝对的问题，我们直接给出否定的答案就行了：不是，不是。

下面，我们还是来认真分析下这个问题。

在回答“科学真的有那么“科学”吗？”之前，我们要先回答另外一个问题：什么是科学？

我们当然可以从百度百科，维基百科，或者是什么其他的百科，甚至是词典上，得到科学的定义。但是，我不想这么做。

  • 首先，我们的大脑并不是这样运作的。每个概念在我们的脑中，当然不是以各种百科或者词典上那样的条文形式来储存和记忆的。我们先是把概念的转化为自己的理解，再记下来，然后才能使用。
  • 其次，我们给任何东西下定义，都不可能是完备的，没有遗漏的，绝对准确的 （记得么？我们刚说过，没有什么东西是绝对的）。凡是用语言或者文字表达出来的定义，它必然首先受制于我们所使用的语言，文字。想想就是呀，我们不可能用语言来表达自我一个这种语言里面并不存在的概念，对吧？

所以，对于“什么是科学？”，我更想给出我自己的一个理解：

所谓科学，其实不过是一套我们用来解释这个世界的复杂的理论体系而已。而且是迄今为止所有理论体系当中，最好用，最完善，最契合实际的一套。

是的，就是这样。
简单，直白，易于理解，揭掉科学的神秘面纱后，我们得以直击科学的本质。这个理解里面，它里面至少包含了这么几层意思：

1. 科学是用来解释现实世界的。
2. 科学是一套复杂的理论体系。
3. 解释世界的理论不止一套。
4. 解释世界的各种理论是有高下之分的。且科学是里面最好的。

从这个角度来看，所谓的封建迷信，宗教，科学，它们本质是其实都是一回事儿。区别仅在于，它们是用来解释这个世界的不同的理论体系而已。

对于这点，我们举例子来加以说明。

关于地球在宇宙中的地位，从最开始的天为盖地为庐，到后来的地心说，再到后来的日心说，到今天的太阳系属于银河系，这样磕磕碰碰一路走过来，我们走的并不容易，有时付出过生命的代价。虽然以今天的眼光来看，“天为盖地为庐”、“日心说”难免显得幼稚、可笑，但在当时，它确实为当时的人们解释了世界的样子。

现在我们都明白，风雨雷电都是很普通的自然现象而已，它们产生的原理，可以用很浅显的物理知识来解释说明。所以，对于封建迷信里面的“风雨雷电是有神仙在操作和指挥” 的鬼神论，我们会感到匪夷所思。但是在当时，这种明显不靠谱的说法，满足了人们对于周遭世界强烈的认知渴望。而且这种鬼神论在当时大行其道，绵延数个世纪之久。

所有宗教的产生初衷，都是为了给人们提供一套解释周遭世界的理论依据，人们可以根据这套方法论来指导他们的生活和生产。经过不断演变和发展之后，宗教（们）才有了提供精神依托的功能。

好了，说完了科学的定义。现在我们来正式答题：讲真，科学真的没有那么“科学”

我们可以从以下方面来考虑：

  1. 我们的认知必然受制于我们的历史发展进程。我们只能说科学是所有用来解释世界的理论里面最成熟的，但离真理还差的远。
  2. 所有的理论都是在不断的发展中完善起来的。我们现在还远没有达到全知全能的程度，所以科学很可能还处于幼儿时期，甚至是婴儿期，那它又怎么可能是完备的呢？
  3. 现在有很多问题，是科学不能解释的，比如我们所谓的超自然现象。
  4. 科学本身的可证伪性，给它自己留下了足够的成长空间。

一句话说清楚就是：科学有可能是错的（全部或者部分，虽然可能性不大），但它会不断发展和进步。

4. 跨越科学,我们看到未来

什么？我们奉为圭臬的科学居然也有可能是错的！？这不科学呀！

是呀，就是这样。并且，我也理解此刻你心中那种千万匹神兽呼啸而过的感受。但真的就是这样。

既然科学在本质是跟封建迷信和宗教是一回事儿，那么我们就很难保证它不会重蹈封建迷信和宗教的覆辙。但是科学比封建迷信和宗教强的一点是，所有的科学理论都有一个必要属性，叫做 可证伪性。

可证伪性是什么呢？所谓可证伪性，说是说：

如果一个主张是可证伪的，则至少在理论上存在一种观测的方法（即使实际上没有进行这项观测也无妨），来表明这个主张不符合重言式的标准（即这个主张不总是真的）。

就是说，科学从骨子里面就承认，它并不一定总是对的。如果我们发现了与科学理论相悖的现象，对相关理论进行修正就行了。

因此，科学必然会随着人类的进步而不断发展。并且，科学不会象日心说替代地心说那样被一个横空出世新理论所替代。更有可能出现的情况是，科学会不断地吸收和吞并那些个新理论，越发地兼容并包，包罗万象。

可以预见的一点是，那时的科学 （如果还叫科学的话） 已经不再是今天的科学可以同日而语的了。它肯定比今天的科学更先进，更完善，更趋于完美——如果还没能达到的话。那时，今天的问题都还再是问题，现在不科学的，也都早已能够被科学解释了。

所以，迷信科学这件事儿呢，并没有太大问题。只是在迷信的科学的同时，我们要知道，今天的科学还不太科学，它将来会更科学就行了。带着不完美上路吧！假以时日，科学一定不会让我们失望的。

至于其他的迷信呢？拉倒吧！

迷信它们，还不如迷信科学呢！因为它们根本就没有科学那么科学的嘛！
