聊聊微信红包那点儿事儿

发表于 2017-06-13 | |

我相信，用微信的各位，对微信红包都不会感到陌生啊？今天，我们就来聊一聊关于微信红包那点儿事儿：什么才是发红包和抢红包的正确姿势！

红包，作为我们古老文化传统的一部分，任凭时光流逝，毅然坚挺地流传到了今天。而微信，作为我们这个时代的旗手，则把发红包和抢红包这个事情，发展到了极致。

我觉得红包的相关的所有问题，概括点儿说＂得体＂二字足矣。

下面我再把自己的意见展开说一下。

1，得体意味着收发有时。 比如，别人讨论问题的时候不宜收，冷场没话题的时候可以发；太早太晚了不宜发，午休时段不宜发，空闲时段随便发(下午六点到九点之间？) ；工作时段不宜发，上班前下班后一小段时间可以发，等等。

2，得体意味着该收时候收，该抢时候抢，该发时候发。 关于这一点，歌里面唱的好呀：该出手时就出手哇，风风火火闯九州哇！啥意思呢？就是说不该出手的时候，千万得忍住了，该出手的时候呢，也千万别客气。

比如我的原则就是，没有参与的话题不抢；发给指定人，指定条件的，看清楚了再抢；抢错了马上退回并致歉；抢过几次之后，一定要发出来，至少也得发一个。道理特别简单：来而不往非礼也。

记住了，＂光抢不发＂和＂平时不冒泡抢红包时候比谁都快＂这两种情况都是要坚决杜绝的。相信我，这样的人，不管去到哪个群体，都不会受人待见的。

红包要能收能发，收发自如。 做到这两点，小伙伴们才能一起愉快地玩耍。

3，得体，就意味着不得让他人和自己难堪，不能扫了大家的兴。

比如说，别人发红包你没抢着，就叫着嚷着让别人再发一个，就很不得体。又比如，抢的少了，或者觉得人家红包发少了，就阴阳怪气地说些个无聊话，不管是发自内心的，还是想抖个机灵，也都是很不得体的。再比如，打肿脸充胖子，有事没事使劲发，而且尽可能往大里发，这样的行为我认为同样是很不得体的。因为你这样做不但给大家造成很大压力，而且万一给自己造成的财务上面的负担，也同样是很不好的。何苦为难自己呢？！

【重要备注：最后这个例子完全不适用于群主和各位豪。敬爱的群主和各位豪大大们，请你们无情地用红包来狠狠砸我吧！直到天荒地老，直到海枯石烂，直到天无棱，地无角，天地合，愿你们的红包雨不曾有过片刻停歇…… 】

4，抢红包发红包的时候，我们一定要有清醒的认识，大家在一个群里面，为了同样的目标，相互学习，相互讨教，相互帮助，大家都是相互平等的。可能有的人暂时厉害一点，有经验一点，能够为其他人提供的帮助多一点，但这些都不会改变人与人之间的平等性。发红包是自发的，自愿的，发的多发的少也应该量力而为。红包这事儿没有谁亏欠谁，也没有谁辜负了谁。抢到了就当是白得的，但要想着适时发出去。没抢到也没有必要懊恼，你要坚信，红包总会有下一波的。

这里澄清一下啊，我并没有要你整天啥事儿不干，就盯着群里抢红包的意思啊，只是想表达：红包这种事，平常心就好了。工作生活才是第一位的。抢红包，娱乐罢了。因为娱乐的小事而耽误了工作和生活这样的大事，那可要不得。

有的人，抢的少了或者没抢到，便出言不逊，大放厥词，更有甚者呼天抢地，满地打滚，口吐白沫，那就更加使不得了。这样做，一来寒了发红包的人的心，二来使你往昔辛苦经营的个使人形象毁于一旦－－如果之前有的话，完全是得不偿失呀。我怕你冷静下来之后，后悔都来不及呀！而且我相信，在你闹过之后，但凡有点儿眼力劲儿的小伙伴，也都会对你退避三舍了吧。

6，坚决抵制一切以抢红包的名义开展各种网络聚众赌博的行为！

5，红包是什么？红包其实就是个润滑剂，红包就是调节气氛，调动情绪，弥合嫌隙，化解尴尬之万灵神药。

是药三分毒，就算是神药也不例外。如何用好红包这个工具，来达到与人愉快地交往的目的，还是看个人修养，觉悟和品性的。俗话说，酒品鉴人品，我看红包亦然。不要小看了抢红包这个小动作，它可以把一个人的人品，修养完整地反映出来。

另外，收发红包这些小原则，不单单只在收发红包时适用，在群里面发消息聊天，平时做人做事时也同样适用。

千言万语汇成一句话：小红包，大名堂，收发需谨慎哪！
